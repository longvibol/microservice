-------------- Create Mongo in Docker : 

mvn clean package

1. Pull from docker hub 

docker pull mvertes/alpine-mongo

2. run mongodb on port 7777 

docker run -p 7777:27017 mvertes/alpine-mongo

3. Connect to MongoCompass 

in mongo compass : mongodb://localhost:7777 = connection  

4. to stop docker 

docker stop ac748dfe7f26
