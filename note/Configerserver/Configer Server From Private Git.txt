--- Refree Service ---

Why we need it ? (we want to change the information in our server :

- Don't want to deploy the serveice again 
- Effect to the system while runing the server 
- some server no need to update 
+ location to put the refresh in each Server (account, loan, card...) 

Step to set up in each service: 

1. Add dependency 

<dependency>
			<groupId>org.springframework.boot</groupId>
			<artifactId>spring-boot-starter-actuator</artifactId>
</dependency>

2. Add anotatoin : @RefreshScope in main app

@RefreshScope
@SpringBootApplication
public class AccountApplication {

	public static void main(String[] args) {
		SpringApplication.run(AccountApplication.class, args);
	}

}

3. In order to refresh without restart service. we setup POST to the service we setup 

POST: http://localhost:8080/actuator/refresh
