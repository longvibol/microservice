Tempo : Use for Tracking log from server to service 

* Step to setup 

1. In grafana Datasource add below to connect from Loki to Tempo (it called derivedFields) 


jsonData:
      httpHeaderName1: "X-Scope-OrgID"
      derivedFields:
        - datasourceUid: tempo
          matcherRegex: "\\[.+,(.+),.+\\]"
          name: TraceID
          url: '$${__value.raw}'

--- in grafana go to datasource ->loki

Navigate to Tempo from Loki 

Connection -> Datasource -> Loki -> Derived filed -> Add -> Namej: TraceID
-> Regex:\[.+,(.+),.+]


Show exmaple log message: [account,hdhdhdhde33333,399398388]
It will extrace TraceID 
-> Click on internal Link => select Tempp

-> Qurey: ${--value.raw}

----------

1. Add datasource to grafana 

apiVersion: 1

deleteDatasources:
  - name: Prometheus
  - name: Loki
  - name: tempo

datasources:
  - name: Prometheus
    type: prometheus
    uid: prometheus
    url: http://prometheus:9090
    access: proxy
    orgId: 1
    basicAuth: false
    isDefault: false
    version: 1
    editable: true
    jsonData:
      httpMethod: GET
      
  - name: Tempo
    type: tempo
    uid: tempo
    url: http://tempo:3100
    access: proxy
    orgId: 1
    basicAuth: false
    isDefault: false
    version: 1
    editable: true
    jsonData:
      httpMethod: GET
      serviceMap:
        datasourceUid: 'prometheus' 

  - name: Loki
    type: loki
    uid: loki
    access: proxy
    orgId: 1
    editable: true
    url: http://gateway:3100
    jsonData:
      httpHeaderName1: "X-Scope-OrgID"
      derivedFields:
        - datasourceUid: tempo
          matcherRegex: "\\[.+,(.+),.+\\]"
          name: TraceID
          url: '$${__value.raw}'
    secureJsonData:
      httpHeaderValue1: "tenant1"
